# MV Competitive Intelligence Dashboard v2

This version separates the dashboard into lightweight pages/buttons:

- Executive
- High Threat Updates
- Viper Competitors
- AccuSense Competitors
- Trident / SF6-Free
- Datasheets & PDFs
- All Intelligence

Upload the contents of this folder to the root of your GitHub Pages repo. Keep `.github/workflows/update-tracker.yml` directly under `.github/workflows/`.

Run the crawler manually from GitHub Actions: **Actions → Update MV Competitive Tracker → Run workflow**.
